# Морозилка iOS App

Нативное iOS приложение для учета домашних заготовок в морозилке, написанное на SwiftUI.

## Требования

- macOS 13.0+
- Xcode 15.0+
- iOS 16.0+

## Установка и запуск

### Шаг 1: Создание Xcode проекта

1. Откройте Xcode
2. Выберите **File → New → Project**
3. Выберите **iOS → App**
4. Заполните поля:
   - **Product Name**: FreezerApp
   - **Team**: Выберите ваш Apple Developer аккаунт
   - **Organization Identifier**: com.yourcompany (замените на свой)
   - **Interface**: SwiftUI
   - **Language**: Swift
   - **Storage**: None (мы используем UserDefaults)
5. Нажмите **Next** и выберите директорию `ios-app`
6. Нажмите **Create**

### Шаг 2: Добавление файлов в проект

После создания проекта, Xcode создаст базовые файлы. Вам нужно:

1. **Удалить** автоматически созданные файлы:
   - `ContentView.swift`
   - Старый файл `FreezerApp.swift` (если создан)

2. **Добавить** все файлы из нашей структуры:
   - В Xcode, кликните правой кнопкой на проект в навигаторе
   - Выберите **Add Files to "FreezerApp"**
   - Выберите все папки: `App`, `Core`, `Features`, `UI`
   - Убедитесь, что выбрано **"Copy items if needed"** и **"Create groups"**
   - Нажмите **Add**

### Шаг 3: Настройка проекта

1. Выберите проект в навигаторе
2. Во вкладке **General**:
   - **Display Name**: Морозилка
   - **Bundle Identifier**: com.yourcompany.FreezerApp
   - **Version**: 1.0
   - **Build**: 1
   - **Minimum Deployments**: iOS 16.0

3. Во вкладке **Info**:
   - Убедитесь, что **Application Scene Manifest** настроен корректно

### Шаг 4: Запуск

1. Выберите симулятор iOS (например, iPhone 15 Pro)
2. Нажмите **Cmd+R** или кнопку **Play**
3. Приложение запустится в симуляторе

### Шаг 5: Запуск на реальном устройстве

1. Подключите iPhone к Mac
2. Разблокируйте iPhone
3. В Xcode выберите ваш iPhone вместо симулятора
4. Нажмите **Cmd+R**
5. На iPhone появится запрос на доверие разработчику:
   - Откройте **Настройки → Основные → VPN и управление устройством**
   - Выберите ваш Apple ID
   - Нажмите **Доверять**
6. Запустите приложение снова

## Структура проекта

```
FreezerApp/
├── App/
│   └── FreezerApp.swift          # Точка входа приложения
├── Core/
│   ├── Models/
│   │   ├── Category.swift        # Модель категории
│   │   └── Item.swift            # Модель заготовки
│   └── Repository/
│       └── DataRepository.swift  # Управление данными (UserDefaults)
├── Features/
│   ├── Categories/
│   │   └── Views/
│   │       ├── CategoryListView.swift    # Список категорий
│   │       └── CategoryFormView.swift    # Форма категории
│   └── Items/
│       └── Views/
│           ├── ItemListView.swift        # Список заготовок
│           └── ItemFormView.swift        # Форма заготовки
└── UI/
    ├── Components/
    │   ├── CategoryCard.swift    # Карточка категории
    │   └── ItemRow.swift         # Строка заготовки
    └── Theme/
        └── Theme.swift           # Цвета и стили

```

## Функциональность

### Категории
- ✅ Создание категорий с иконками и цветами
- ✅ Редактирование категорий
- ✅ Удаление категорий
- ✅ Подсчет количества заготовок в категории
- ✅ Предустановленные иконки (30 эмодзи)
- ✅ Предустановленные цвета (8 цветов)

### Заготовки
- ✅ Добавление заготовок со всеми полями
- ✅ Редактирование заготовок
- ✅ Удаление заготовок (свайп влево)
- ✅ Быстрое изменение количества упаковок (+/-)
- ✅ Индикатор срока годности (свежее/истекает/просрочено)
- ✅ Поле для заметок
- ✅ Поиск по названию и заметкам
- ✅ Фильтрация по номеру полки

### Хранение данных
- Локальное хранение в UserDefaults
- Автоматическое сохранение при изменениях
- Инициализация с примерами данных при первом запуске

## Технологии

- **SwiftUI** - Декларативный UI framework
- **UserDefaults** - Локальное хранение данных
- **Codable** - Сериализация моделей
- **@StateObject / @EnvironmentObject** - Управление состоянием
- **NavigationStack** - Навигация
- **Form / Sheet** - Модальные формы
